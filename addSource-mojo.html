gplus:addSource

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:1.0-beta-2:addSource

Description:

This mojo adds Groovy sources to the project's sources.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
sources FileSet[] - The Groovy source files (relative paths). Default: "${project.basedir}/src/main/groovy/**/*.groovy"
testSources FileSet[] - The Groovy test source files (relative paths). Default: "${project.basedir}/src/test/groovy/**/*.groovy"
