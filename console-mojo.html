gplus:console

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:1.1-SNAPSHOT:console

Description:

Launches a Groovy console window bound to the current project. Note that this mojo requires Groovy >= 1.5.0.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.1.
