gplus:execute

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:1.0-beta-2-SNAPSHOT:execute

Description:

Executes Groovy scripts (in the pom or external).

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.

Required Parameters

Name Type Since Description
scripts String[] - Groovy scripts to run (in order). Can be an actual Groovy script or a java.net.URL URL to a Groovy script (local or remote).

Optional Parameters

Name Type Since Description
continueExecuting boolean - Whether to continue executing remaining scripts when a script fails.
Default value is: false.
sourceEncoding String - The encoding of script files.
Default value is: ${project.build.sourceEncoding}.

Parameter Details

continueExecuting:

Whether to continue executing remaining scripts when a script fails.
  • Type: boolean
  • Required: No
  • Default: false

scripts:

Groovy scripts to run (in order). Can be an actual Groovy script or a java.net.URL URL to a Groovy script (local or remote).
  • Type: java.lang.String[]
  • Required: Yes

sourceEncoding:

The encoding of script files.
  • Type: java.lang.String
  • Required: No
  • Default: ${project.build.sourceEncoding}
