gplus:testGroovydoc

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:1.0-beta-2-SNAPSHOT:testGroovydoc

Description:

Generates Groovydoc for the test sources. Note that this mojo requires Groovy >= 1.6.2.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • The goal is thread-safe and supports parallel builds.
  • Invokes the execution of the lifecycle phase generate-test-sources prior to executing itself.

Optional Parameters

Name Type Since Description
displayAuthor boolean - Whether to display the author in the generated Groovydoc
Default value is: true.
docTitle String - Page title
Default value is: Groovy Documentation.
footer String - Page footer
Default value is: Groovy Documentation.
groovydocOutputDirectory File - Location for the generated API docs
Default value is: ${project.build.directory}/gapidocs.
header String - Page header
Default value is: Groovy Documentation.
overviewFile File - HTML file to be used for overview documentation
scope String - Scope to generate Groovydoc for, should be "public", "protected", "package", or "private"
Default value is: private.
sources FileSet[] - Groovy source files (relative paths). Default: "${project.basedir}/src/main/groovy/**/*.groovy"
stylesheetEncoding String - Encoding of stylesheetFile
Default value is: UTF-8.
stylesheetFile File - Stylesheet file (absolute path) to copy to output directory (will overwrite default stylesheet.css)
testGroovydocOutputDirectory File - Location for the generated test API docs
Default value is: ${project.build.directory}/testgapidocs.
testSources FileSet[] - Groovy test source files (relative paths) Default: "${project.basedir}/src/test/groovy/**/*.groovy"
windowTitle String - Window title
Default value is: Groovy Documentation.
